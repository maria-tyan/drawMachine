# DrawMachine

*Данная рисовалка выполнена в рамках лабораторной работы*

## Возможности рисовалки:

Ниже перечислены все возможные активности рисовалки. Для каждой из них реализован метод отправки ланных на сервер через emit библиотеки Socket.IO. 
Однако, для pen и erraser таких методов нет, так как их работа основана на активности типа line, а дальнейшая обработка подобных событий ведется на клиентской сторне.

__action = 'rectangle'__ - позволяет рисовать прямоугольники 
__action = 'arrow'__ - позволяет рисовать стрелки, направленные от начального положения курсора к конечному
__action = 'circle'__ - позволяет рисовать круги
__action = 'line'__ - позволяет рисовать прямые линии
__action = 'pen'__ - позволяет рисовать произольные линии, кривая будет следовать за движением пользовательского манипулятора
__action = 'erraser'__ - позволяет стирать все, что было нарисованно, кривая будет следовать за движением пользовательского манипулятора
__action = 'text'__ - позволяет писать текст

Так же в рисовалке реализован выбор различных цветов для рисования и изменения тощины кисти.
